10月 - icpc / 專題 / 電算中心

  • 專題開始跟教授開會，想parasite的主題。
  • 拉icpc隊友學宇跟文獻還有李卓報了Nasa的hackathon，原本只是想水贈品，但是後來肝了半天生出一個網站，最後得了台中站佳作。
  • world quant alphathon結束，中央第一名(雖然這比賽根本是先卡位先贏)，拿了1000美金跟當顧問。
  • 跟隊友去師大比ncpc決賽，一大早去租irent從中央開到師大；結果我們打得超爛...晚上在師大吃順便討論icpc怎麼辦。
  • 考了cpe，題目變難 + 中場離開去開專題會，ac 2題而已 這比我大一第一次打還爛== 完全在浪費時間。
  • 電算中心開始跟學長開會，還是很佩服學長作出那個網站，然後一堆資安的漏洞我都不太會修，trace code的能力也要加強。
  • 跟隊友去桃園體育館打icpc，拿了銅牌，算是圓了大一的夢。心得文
  • 機率期中考，意料之中的成績，就是不夠認真。
  • 計算機網路跟民法期中考，尚可。 不過我這學期修民法真的很意料之外。
  • 買了chatGPT plus

11月 - wq / 曼谷 / 人工智慧

  • 第一次去world quant辦公室，那邊的researcher教我們建立自動化產alpha的pipieline，用genetic algorithm，很有趣。但我現在只完成串接api登入跟生物驗證用line bot寄給我(還特地租了一個vps)。
  • 第三周全校運動會，只有一有課，過得很爽。
  • 去清華聽了kronos說明會，感想就是相比其他外國量化公司駐點台灣，kronos總部就在台北，對台灣投入的資源最多，相比其他量化公司，門檻也低了那麼一咪咪；kronos是作虛擬貨幣高頻交易(類似造市商)，對每個員工程式能力要求都比較高，python算是基本，infra team的話，c++也要會，程式大概佔70%，wq則是數學金融知識跟程式比例大概各半。
  • 完成人工智慧的作業2，真的花超多時間。叫我們用RL操作外匯，原本用policy gradient訓練到最後變成全買歐元，於是又去作了特徵工程增加一堆特徵再把網路改成lstm模型用dqn，結過score反而變低==，最後交出去的一開始全買歐元的模型...。
  • 家人參加研討會的緣故，去了一趟泰國曼谷。(這應該之後會再打一篇心得) (結果沒打)

12月 - 期末 / 其實好像沒做甚麼

  • 南港唐吉軻德、tsutaya，人好多。
  • 台大天數圖再訪，真是個好地方。
  • 去參加world quant Q4的聚會，上台報告的人分享的方法都很厲害。wq新手禮包get。
  • 信義誠品關了，改天去看看松菸24小時誠品。
  • 買了airpods pro 2 usbc。我的第一個降躁耳機。現在戴下來大概兩小時，耳多就會開始痛。
  • 期末轟炸。
  • 去台大nasa hackathon的頒獎典禮，神奇的體驗。
  • 期待已久的導聚，吃千葉火鍋，評價3/5星。完後去對面家樂福鬼混。
  • 思考2024該幹嘛。

結論

今年相比大一大二應該可以說充實了不少? 做了許多奇奇怪怪的事，拿到電算中心職缺 / icpc 銅牌(寫在天燈果然有用) / 變成wq顧問 / 在日本開車! / 一個人在曼谷晃 / 開車載同學玩 / 搬進曦望居...
