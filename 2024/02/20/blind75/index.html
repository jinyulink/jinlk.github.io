Array

1. Two Sum

給一個整數陣列，以及一個目標值，找出兩個數字的index，使得這兩個數字的和等於目標值。

暴力解: 兩層迴圈，時間複雜度O(n^2)。 / Hashmap: O(n)。

Two Sum II 如果是sorted array，binary search (O(nlogn))，two pointer (O(n))。

面試有可能先出sorted array的版本，follow up再出unsorted array的版本。或是在出一個3sum。

2. Best Time to Buy and Sell Stock

給一個整數陣列，代表每天的股價，找出最大利潤。

暴力解: 兩層迴圈，時間複雜度O(n^2)。 / 一次迴圈: 維護一個變數紀錄前i天的最小值，每次更新最大利潤，O(n)。

3. Contains Duplicate

給一個整數陣列，判斷是否有重複的數字。

Hashmap: O(n)。

4. Product of Array Except Self (待)

給一個整數陣列，計算除了自己以外的所有數字的乘積。

5. Maximum Subarray

給一個整數陣列，找出和最大的subarray。

暴力解: 兩層迴圈，時間複雜度O(n^2)。

dp: f[i] = max(f[i-1] + nums[i], nums[i])，O(n)，這樣空間O(n)。

Kadane's algorithm: O(n)，基本上就是dp優化版本，因為每次我們只需要f[i-1]，實際上可以不用開到一整個陣列去存，維護一個變數就好了，這樣空間O(1)。維護curMax代表以i為結尾的subarray的最大和，每次更新答案。

6. Maximum Product Subarray

給一個整數陣列，找出和最大的subarray的乘積。

想法跟上一題差不多，用kadane's algorithm，可以達到O(n)，空間O(1)，但是要多維護一個curMin，因為負數的存在，最大的乘積可能是負數乘上負數。也就是說維護curMax和curMin，每次更新答案。

7. Find Minimum in Rotated Sorted Array

給一個sorted array，這個array被rotate了，找出最小的數字。 eg. [4,5,6,7,0,1,2] -> 0

Naive: 掃一遍，O(n)。

Binary search: O(logn)。
